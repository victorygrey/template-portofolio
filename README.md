## ğŸš€ Cara Menggunakan

### 1. Clone atau Download Repository

```bash
git clone https://github.com/your-username/template-portofolio.git
cd template-portofolio
```

Atau download sebagai ZIP dan ekstrak ke folder yang diinginkan.

### 2. Buka di Browser

Buka file `index.html` langsung di browser Anda, atau gunakan live server:

**Menggunakan VS Code:**
- Install extension "Live Server"
- Klik kanan pada `index.html` â†’ pilih "Open with Live Server"

**Menggunakan Python:**
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

Kemudian buka `http://localhost:8000` di browser.

### 3. Kustomisasi Konten

Edit file `index.html` untuk mengubah konten sesuai kebutuhan Anda:

- **Informasi Pribadi**: Ganti nama, deskripsi, dan informasi di section Hero dan About
- **Skills**: Update tabel skills di section Skills
- **Projects**: Edit project cards di section Projects
- **Experience**: Update pengalaman kerja di section Experience
- **Contact**: Ubah email dan link sosial media di section Contact dan Footer
- **Meta Tags**: Update meta tags di `<head>` untuk SEO (og:url, og:image, dll)

### 4. Kustomisasi Styling

Edit file `style.css` untuk mengubah warna, font, spacing, dan styling lainnya sesuai brand Anda.

### 5. Ganti Gambar

- Ganti `img/logo.png` dengan logo Anda
- Ganti `img/banner.png` dengan banner/hero image Anda
- Pastikan ukuran gambar sudah dioptimasi untuk performa website
